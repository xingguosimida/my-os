# my-os
操作系统实验
# How to run it
执行下面脚本以安装必要的依赖
```sh
brew install bison
brew install flex
brew install gmp
brew install mpc
brew install mpfr
brew install texinfo
brew install cloog
brew install isl
brew install binutils
brew install gcc
brew install i686-elf-gcc
brew install nasm
brew install qemu
```

Run `./auto/run`.
